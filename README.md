# Telegram Pixel Field (MVP)

Игра в Telegram: большой пиксельный мир, игроки спавнятся на краю карты, красят клетки рядом с собой, получают очки и монеты, находят лут (чаще в центре) и покупают бусты.

## Что внутри

- `backend/` — FastAPI сервер + SQLite база
- `bot/` — Telegram bot (aiogram) с кнопкой запуска WebApp
- `webapp/` — фронт (Canvas) для игры

## Быстрый старт (Windows / PowerShell)

1) Создайте виртуальное окружение и установите зависимости:

```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

2) Создайте `.env` рядом с `.env.example` и заполните:

- `BOT_TOKEN` — токен бота от BotFather
- `WEBAPP_URL` — URL веб-приложения (локально можно оставить как в примере)

3) Запустите backend:

```bash
uvicorn backend.main:app --reload --host 0.0.0.0 --port 8000
```

Откройте `http://localhost:8000/webapp/` в браузере для быстрой проверки.

4) Запустите бота (в другом терминале):

```bash
python -m bot.bot
```

## Игровые правила (MVP)

- **Карта**: 2000×2000 клеток. Хранится “разреженно” (в БД только то, что покрашено/содержит бонус).
- **Спавн**: на случайной точке по краю карты.
- **Покраска**: можно красить только клетки на расстоянии 1 (по Манхэттену) от позиции игрока. Покраска перемещает игрока в эту клетку.
- **Награды**: за каждую новую покрашенную клетку игрок получает монеты и очки. В центре карты шанс лута выше.
- **Лут**: выпадает при покраске (чаще ближе к центру). Можно открыть и получить монеты или временный буст.
- **Бусты за монеты**: магазин позволяет купить временные усиления (например, больше награда, ускорение, расширение дистанции покраски).
